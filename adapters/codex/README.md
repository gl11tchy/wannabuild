# Codex Adapter

Primary WannaBuild adapter.

Use the repo directly:
- `AGENTS.md`
- `skills/`
- `agents/`
- `scripts/`

Install:

```bash
./scripts/install-codex-skill.sh
```

Then restart Codex and invoke:

```text
$wannabuild
```
