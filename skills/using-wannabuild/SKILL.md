# Using WannaBuild

Use this skill when the user wants to work with WannaBuild itself or asks how to start.

## What To Do

Tell the agent to use `$wannabuild` for the actual workflow.

WannaBuild runs:

1. Discover
2. Plan
3. Implement
4. Review
5. QA
6. Summary

## Start Prompt

If the user needs a starting point, suggest:

```text
$wannabuild

I want to build:
[describe the feature or project]
```
