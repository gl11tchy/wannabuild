# Requirements

## Goal

Demonstrate a completed WannaBuild run with enough evidence to resume, review, QA, and summarize without guessing.

## Acceptance Criteria

- The workflow state reaches `public_stage: "summary"` and `workflow_status: "complete"`.
- Requirements, design, tasks, checkpoints, review verdicts, and QA summary exist.
- Review verdicts are passing.
- Integration tester evidence covers the acceptance criteria.
